# data_robust_hub_interdiction
This repository contains data used for Robust Hub Interdiction manuscript
The first two dimensional matrix is the nominal demand between the node pairs
The second two dimensional matrix is the distance between the node pairs
The third one dimensional matrix is the hubs located when p = 7
The fourth one dimensional matrix is the hubs located when p = 10
The fifth one dimensional matrix specifies tau_i values for each origin node i
The sixth one dimensional matrix specifies the multiplicative factor for the nominal demand when setting is low
The seventh one dimensional matrix specifies the multiplicative factor for the nominal demand when setting is medium
The eighth one dimensional matrix specifies the multiplicative factor for the nominal demand when setting is high
